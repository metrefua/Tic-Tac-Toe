# Tic-Tac-Toe

## Project Overview

This project is a modern Tic Tac Toe web application designed with both **Single Player (vs Computer)** and **Two Player (vs Friend)** modes.

The goal of the project is to create a responsive and user-friendly game experience.

The application includes player setup, score tracking, restart and reset controls, and smart computer gameplay.

---

## Features

### Game Modes

* Single Player mode (Player vs Computer)
* Two Player mode (Player vs Player)

### Gameplay Features

* Smart computer move logic
* Winner detection
* Draw detection
* Turn status display
* Winner popup modal

### Score System

* Player X score tracking
* Player O / Computer score tracking
* Draw counter

### Controls

* Start Game
* Restart / Next Round
* Reset / New Match

### UI Features

* Responsive design for mobile and desktop
* Clean modern interface
* Interactive buttons and hover effects
* Player name input fields

## How the Game Works

1. Select game mode (Single Player or Two Player)
2. Enter player name(s)
3. Click **Start Game**
4. Players take turns placing X and O
5. The system checks for:

   * Winner
   * Draw
6. Scores update automatically
7. Use Restart for next round or Reset for a fresh match
